# Rental Drone Website (GitHub Pages)
Template siap upload ke GitHub Pages.

## Cara Deploy
1. Buat repository baru di GitHub.
2. Upload seluruh isi folder `drone_rental_site`.
3. (Opsional) Ganti placeholder link `https://wa.me/qr/RFULN3DY7HULF1`, `https://www.tiktok.com/@pilotdronebws?_r=1&_t=ZS-91unFbydtfZ`, `https://www.instagram.com/rdwansyarf?igsh=MWloN2xjN3J5NWNmbg==`.
4. Settings → Pages → Pilih branch `main` dan folder `/root`.
5. Tunggu beberapa menit, lalu akses: `https://username.github.io/repo-name`
